# Laboratorio 52: Control por voz de un semáforo modular

## Objetivo

Integrar reconocimiento de voz offline (Vosk) con Tkinter para controlar un semáforo industrial tricolor simulado.

## Comandos
- "Enciende roja"
- "Enciende amarilla"
- "Enciende verde"
- "Apagar todas"
- "Salir"
